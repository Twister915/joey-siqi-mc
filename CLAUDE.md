# Siqi Minecraft Plugin

A Bukkit/Paper Spigot plugin for Minecraft 1.21.10.

## Project Overview

This is a small personal plugin with tiny quality-of-life modifications to vanilla Minecraft. The goal is to make the game more enjoyable for Joey and Siqi to play together.

## Technical Details

- **Platform**: Paper Spigot (Bukkit API)
- **Minecraft Version**: 1.21.10
- **Build System**: Gradle

## Development

Build the plugin:
```bash
./gradlew build
```

The compiled JAR will be in `build/libs/`.
